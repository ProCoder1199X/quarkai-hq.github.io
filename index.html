WHAT WE'RE BUILDING

EASiSCRIPTX

ACTIVE

High-performance domain-specific language for AI/ML workflows

EasiScriptX (ESX) is a declarative DSL that simplifies AI model definition, dataset management, and training routines. Built for both hobbyists and enterprise workflows with minimal resource overhead.

Features advanced tensor operations, distributed training (multi-GPU/multi-node), ONNX interoperability, custom loss functions, and autonomic optimization including model compression and multi-agent tuning.

Supports parameter-efficient fine-tuning (LoRA), FlashAttention-2, mixed-precision training (BF16/FP16), pipeline parallelism, and energy-aware scheduling.

Built with C++20, Boost, Eigen, ONNX Runtime, OpenMPI. MIT licensed, v1.0 available now.
