DSL ML/AI v1.0

EASISCRIPTX

A high-performance domain-specific language designed for AI/ML workflows. Simplifies model definition, datasets, and training with declarative syntax.

Features advanced tensor operations, distributed training (multi-GPU/multi-node), ONNX/PyTorch integration, parameter-efficient fine-tuning (LoRA), FlashAttention-2, mixed-precision training, and pipeline parallelism.

Built with C++20, Boost, Eigen, and CUDA. MIT licensed. Reduces memory and energy usage for both enterprise and hobbyist workflows.

Neuro-Symbolic Multi-Agent Flagship

CONGMING-AI

An experimental, modular neuro-symbolic AI framework with Multi-Agent System architecture, focused on advanced reasoning capabilities.

Successfully modified 8+ code files autonomously. Capable of choosing random topics, learning via internet (data ingestion), and applying stored knowledge contextually.

Benchmarks: Congming-2 achieved 43% pixel accuracy and ~30% task success on ARC-1. Congming-3 reached ~57% pixel accuracy and 18% task success on ARC-AGI2 across ~200 tasks.
